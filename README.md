# 0408_quiz-bite

A simple general knowledge quiz web app built with React and Vite.

## Features
- 5 multiple-choice trivia questions
- Instant correct / incorrect feedback
- Final score summary
- Restart and replay support
- Ready for GitHub Pages deployment

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## GitHub Pages
This project is configured to deploy automatically with GitHub Actions.
